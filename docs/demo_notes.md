# SkyLake — Demo Notes

> Notes for presenting this project to recruiters, hiring managers, or peers.

## Key Talking Points
- End-to-end data engineering: ingestion → storage → transformation → serving
- Medallion architecture: raw → bronze → silver → gold
- Real government data source (NOAA AWC) — no mocked or synthetic data
- Production patterns: Dagster orchestration, dbt modeling, Soda Core quality checks
- Fully reproducible: Docker Compose one-command startup
